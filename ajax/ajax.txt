Ajax(Asynchronous Javascript and XML)
- 비동기적인 웹 애플리케이션 제작을 위해 사용
- XMLHttpRequest() - 예전방식
- jQuery 방식
- fetch() / axios() - 요즘방식

- 비동기적으로 네트워크를 통해서 서버의 리소스(파일-json, html, txt)를 취득

- JSON(Javascript Object Notation)은 다양한 시스템이 데이터를 교환할 수 있는 표준화되고 효율적인 방법을 제공하는 경량 데이터 교환 형식
    => 자바스크립트 객체와 유사함 : {name : "hong", age : 15}
    -> json : {"name" : "hong", "age" : 15}

json <===> 자바스크립트 객체

http 요청에 대한 응답
응답 코드 (http 상태 코드)

200(성공 응답) : ok
403(권한 없음) : 클라이언트에게 권한 없음(일반 사용자가 어드민 사이트 요청)
404(Not Found) : 못찾음(요청이 잘못됐음)
500(Internal Server Error) : 서버 오류(서버가 잘 작동되기를 기다림)

openAPI
- 개방형 데이터 제공
- 시크릿 키를 발급받은 대상자들에게 데이터 오픈
- 네이버, 다음, 구글, 공공데이터 포털, 서울 열린데이터광장......
- 특정한 규칙을 이용해서 데이터 요청

searchDailyBoxOfficeList.json?key=82ca741a2844c5c180a208137bb92bd7&targetDt=20241007

?key=value&key=value ==> 늘 같은 방식

?key=발급받아야 하는 키&targetDt=날짜지정
